# 📸 Assets — Bernardi Arquitetura e Engenharia

## Fotos obrigatórias (substituir placeholders)

Coloque as imagens nesta pasta com os nomes abaixo:

### Hero
| Arquivo | Descrição | Dimensão recomendada |
|---|---|---|
| `hero-obra.jpg` | Foto principal — fachada ou interior bem iluminado de obra concluída | 1600×1000px mínimo |

### Portfólio (mínimo 6, máximo 12)
| Arquivo | Projeto | Categoria |
|---|---|---|
| `portfolio-01.jpg` | Residência unifamiliar | Residencial |
| `portfolio-02.jpg` | Residência térrea | Residencial |
| `portfolio-03.jpg` | Espaço comercial | Comercial |
| `portfolio-04.jpg` | Galpão industrial | Comercial |
| `portfolio-05.jpg` | Regularização residencial | Regularização |
| `portfolio-06.jpg` | Sobrado residencial | Residencial |

> Para adicionar mais projetos: edite o array `projects` em `src/sections/Portfolio.tsx`

### Dimensões recomendadas para portfólio
- **Resolução:** mínimo 800×600px (proporção 4:3)
- **Formato:** JPG otimizado (80% qualidade)
- **Tamanho por arquivo:** máximo 300KB (use TinyJPG ou Squoosh para otimizar)

## Logo (colocar em /public)
| Arquivo | Descrição |
|---|---|
| `favicon.svg` | Favicon 32×32px (SVG preferível) |
| `logo.svg` | Logo da empresa em SVG (usado no header se disponível) |
| `og-image.jpg` | Imagem Open Graph — 1200×630px (para compartilhamento em redes sociais) |

## Depoimentos
Edite o array `testimonials` em `src/sections/Testimonials.tsx` com os 3 melhores
reviews reais do Google, incluindo o nome completo do cliente e o contexto do projeto.
